## 核心设计

### 1. ServiceRegistry 模式
解决循环依赖：服务在 `app.py` 创建，路由在 Blueprint 中访问。

```python
# app.py
services.register('memory_service', MemoryService())
app.services = services

# 路由中
memory_service = current_app.services.memory_service
```

### 2. Socket.IO 跨 Blueprint
将 `socketio` 实例挂载到 `app`：

```python
# app.py
app.socketio = SocketIO(app, cors_allowed_origins="*")

# 路由中
current_app.socketio.emit('processing_status', {...})
```

### 3. 路径保持不变
Blueprint 使用 `url_prefix` 保持所有原有 API 路径。
