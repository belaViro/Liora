# MemoryWeaver 项目进度

## 概述
基于 Flask + 大模型的多模态个人记忆网络系统。

## 技术栈
- **后端**: Flask, Flask-SocketIO, FAISS
- **前端**: 原生JS, D3.js v7, html2canvas
- **AI**: MiniMax API (兼容OpenAI格式)
- **存储**: JSON文件 + FAISS向量索引

---

```bash

python app.py

```
