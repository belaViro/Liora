# Liora 记忆网络设计

## 当前问题
- 只是静态知识图谱（节点+边）
- 缺少"记忆"的动态特性
- 用户感受不到"回忆"的过程

## 记忆网络的核心特性

### 1. 记忆的鲜活度（Decay & Reinforcement）
```
记忆活跃度 = f(时间衰减, 回忆次数, 情感强度)

- 时间越久 → 越淡（除非经常回忆）
- 反复回忆 → 强化（抗遗忘）
- 情感强烈 → 深刻（初恋效应）
```

### 2. 自由联想（Associative Recall）
```
输入：一个触发点（关键词/图片/声音）
↓
激活相关记忆节点
↓
扩散激活（类似人脑联想）
↓
返回记忆链
```

### 3. 时空情境（Contextual Memory）
```
- 时间线：线性回忆
- 空间地图：地点关联
- 情感色彩：快乐/悲伤记忆
- 人物视角：不同人眼中的同一事件
```

### 4. 记忆的可追溯性
```
每个知识都链接到原始记忆片段
- 可查看：从哪段文字/哪张图片提取
- 可验证：原文上下文
- 可修正：更新错误认知
```

## 实现方案

### 阶段1：记忆活跃度系统
- 每个记忆有"活跃度分数"
- 时间衰减 + 回忆强化
- 可视化：节点大小/透明度随活跃度变化

### 阶段2：联想回忆引擎
- 输入触发词 → 向量搜索激活节点
- 沿边扩散（BFS/DFS）
- 返回"记忆链"（A→B→C）

### 阶段3：时空导航
- 时间线视图
- 地图视图（地点聚合）
- 情感过滤器

### 阶段4：溯源与验证
- 点击实体 → 显示来源记忆
- 原文高亮
- 置信度/争议度标记
